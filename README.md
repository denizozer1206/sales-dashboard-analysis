Sales Dashboard Analysis

This project focuses on analyzing sales data using both Python and Excel to uncover key business insights.

Project Overview

In this project:

* Data was cleaned and prepared using Python (Pandas)
* Sales trends and insights were analyzed
* An interactive dashboard was created in Excel

Dashboard Insights

* Technology is the top-performing category
* West region generates the highest sales
* Sales vary significantly across sub-categories

Tools Used

* Python (Pandas)
* Excel (Pivot Tables & Dashboard)

Project Files

* github1.ipynb → Data analysis using Python
* temiz.xlsx → Cleaned dataset
* dashboard.png → Final dashboard

Conclusion

This project demonstrates how raw data can be transformed into meaningful insights using Python and Excel.
